# Ruby Client Test

This is an ruby client app for api test.

## Start API Server 

		ruby lib/server.rb

## Create a user

		ruby client/post_user.rb	 

## Retrieve a user

		ruby client/retrieve_user.rb

## Retreive all users

		ruby client/retrieve_users.rb

## Update a user

		ruby client/update_user.rb

## Delete a user

		ruby client/delete_user.rb
